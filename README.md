# Stocks - 台股即時監控工具

這是一個輕量化的台股即時看盤工具，專為需要快速追蹤特定股票資訊的使用者設計。

---

## 主要功能
* **即時行情更新**：自動追蹤台股最新價格、漲跌幅及成交量。
* **自訂清單**：可隨時新增或刪除感興趣的股票代號，清單將自動儲存。
* **自動縮放介面**：動態調整視窗大小，保持介面簡潔且不佔空間。
* **盤中自動更新**：在開盤期間（09:00 - 13:40）自動定時抓取最新資料。

---

## 使用說明
本工具目前已針對各平台進行優化，使用者只需直接執行下載的 **可執行檔 (.exe / .app)** 即可使用，無需安裝 Python 環境或進行任何繁瑣的程式碼設定。

1. **開啟程式**：啟動後會自動載入上次儲存的股票清單。
2. **新增股票**：在輸入框輸入 4 至 6 碼股票代號（例如：2330），點擊「添加股票」。
3. **刪除股票**：在輸入框輸入代號後點擊「刪除股票」，系統會自動重新整理列表。
